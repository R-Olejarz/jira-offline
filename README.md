# Jira Offline

A lightweight offline task manager for developers, built with .NET 9 and Avalonia UI.

The application stores tasks locally and does not require an internet connection or external account.

## Screenshot

![Jira Offline main view](docs/images/mainview.png?raw=true)

## Technology Stack

- .NET 9
- C#
- Avalonia UI 12.0.5
- Local JSON storage

## Requirements

- .NET 9 SDK
